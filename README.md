# ReservationService

To set up the environment NetBeans so that the database can be accessed, go to services -> databases, right click on "Java DB", and select "properties". Set the "Java DB Installation" folder to be the /database/db folder in this repository.  Set the Database Location to be the /database folder in this repository. Click OK.  Then, right click on "Java DB" again, and this time, select "Start Server".  You should be good to go!
